Servidores y clientes UDP y TCP con Python

Estos ejemplos implementan un servicio muy simple llamado upper. Los servidores upper devuelven al cliente la cadena que se le envía (como el servicio echo) pero transformada a mayúsculas.

Dado que su objetivo es docente, se han sacrificado los controles de errores (tal como las excepciones) en favor de la legibilidad y el tamaño. Una implementación correcta debería añadir dichos mecanismos.
